## Instructions to Run the Tests

1. Open the terminal.

2. Navigate to the test directory:
   ```bash
   cd cypress_Cucumber_Test
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. To view results in the terminal:
   ```bash
   npx cypress run
   ```

5. To open the Cypress UI for results:
   ```bash
   npx cypress open
   ```
```
